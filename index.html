HTML & the Birth of the World Wide Web

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages.

HTML was invented by Tim Berners-Lee in 1991 while he was working at CERN, the European Organization for Nuclear Research.

Tim Berners-Lee also created the first web browser and the first website. This marked the beginning of the World Wide Web (WWW), which changed how people share and access information globally.

Thanks to his work, the internet became more usable and accessible to everyone around the world.
